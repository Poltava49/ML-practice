Этот набор данных содержит информацию об эпизодах популярных подкастов, включая метаданные, жанры, характеристики ведущих и гостей, а также данные о поведении слушателей. Основной целью может быть анализ факторов, влияющих на продолжительность прослушивания.

Описание Фичей
--------------

Podcast_Name
Тип: строка
Значения: Названия популярных подкастов.

Episode_Title
Тип: строка
Значения: Названия конкретных эпизодов подкастов.

Episode_Length
Тип: число с плавающей точкой (в минутах)
Значения: Длительность эпизода в минутах. Примеры: 5.0, 10.0, 30.0, 45.0, 60.0, 90.0.

Genre
Тип: строка
Значения: Жанр подкаста. Возможные значения: "Technology", "Education", "Comedy", "Health", "True Crime", "Business", "Sports", "Lifestyle", "News", "Music".

Host_Popularity
Тип: число с плавающей точкой (от 0 до 100)
Значения: Оценка популярности ведущего по шкале от 0 до 100. Примеры: 50.0, 75.0, 90.0.

Publication_Day
Тип: строка
Значения: День недели, в который был опубликован эпизод. Примеры: "Monday", "Tuesday", ..., "Sunday".

Publication_Time
Тип: строка
Значения: Время публикации. Возможные категории: "Morning", "Afternoon", "Evening", "Night".

Guest_Popularity
Тип: число с плавающей точкой (от 0 до 100)
Значения: Оценка популярности гостя эпизода. Примеры: 20.0, 50.0, 85.0.

Number_of_Ads
Тип: целое число
Значения: Количество рекламных вставок в эпизоде. Примеры: 0, 1, 2, 3.

Episode_Sentiment
Тип: строка
Значения: Общая эмоциональная окраска контента эпизода. Возможные значения: "Positive", "Neutral", "Negative".

Listening_Time
Тип: число с плавающей точкой (в минутах)
Значения: Фактическая средняя продолжительность прослушивания эпизода. Является целевой переменной.
